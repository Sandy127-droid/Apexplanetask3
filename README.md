This project is a practical implementation of advanced web development concepts using HTML, CSS, and JavaScript. It demonstrates responsive web design through CSS media queries, ensuring a seamless experience across mobile, tablet, and desktop devices. The project also includes an interactive quiz that uses JavaScript for dynamic user interaction and instant feedback. Additionally, it integrates a public API to fetch and display real-time data dynamically on the webpage, showcasing the use of asynchronous programming with the Fetch API. This project helps strengthen front-end development skills by combining responsive layouts, DOM manipulation, event handling, and API integration into a single user-friendly application.
